# Инициализация GPS приёмника
- Cold start 1-5 минут
- Warm start 30 - 60 секунд
- Hot start 2-10 секунд

# Приёмник может выдавать данные даже до фиксации позиции
Фиксация - это состояние когда GPS приёмник определил своё местоположение
- для 2D - фиксации (широта, долгота) нужно минимум 3 спутника
- для 3D -фиксации  (широта, долгота, высота) минимум 4 спутника
В $GPGGA есть поле fix quality:
- 0 - нет фиксации
- 1 - стандартная GPS-фиксация
- 2 — DGPS‑фиксация (с дифференциальными поправками, точнее).
- 4 — RTK‑фиксация (сантиметровая точность).

# Факторы качества связи
1. Количество спутников (8 - 12 устойчивая фиксация)
2. Геометрия спутников (GDOP/PDOP) показатель качества расположения спутников на небосводе
(оптимально когда распределены по разным направлениям)
4. Препятствия и условия окружающей среды
5. Атмосферные условия.
6. Многолучевость (переотражение сигнала)
