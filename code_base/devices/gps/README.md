# GPS

Модуль для работы с GPS‑устройствами, включающий парсер данных, базу GPS приёмников, автодетектор порта и примеры использования.

## Возможности

- 📡 Получение и обработка GPS‑данных
- 🗂️ Хранение координат в удобных структурах
- 🔍 Автоматическое определение порта подключения
- 🖥️ Визуализация данных через GUI‑виджет
- ⚡ Минималистичные примеры для быстрого старта
