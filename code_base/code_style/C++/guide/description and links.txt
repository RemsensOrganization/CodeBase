1. Качаем и установливаем .exe файл(LLVM-18.1.6-win64.exe), ссылка: https://github.com/llvm/llvm-project/releases/tag/llvmorg-18.1.6 .
2. Включаем необходимые модули, см. скрин_1.
3. Появится модуль "Стилизатор", выбираем необходимые параметры, см. скрин_2, скрин_3. Файл(.clang-format) со стилем кода должен находится в папке проекта.
4. Можно включить чтобы стиль применялся при сохранении файла или на горячую клавишу, см. скрин_4, скрин_5 соответсвенно.
