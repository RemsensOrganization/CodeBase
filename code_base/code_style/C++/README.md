# C++
Правила и утилиты для автоформатирования кода С++ в проектах CodeBase

## Настройка_и_установка
1. Включить необходимые модули, см. images/image_1.
2. Появится модуль "Стилизатор", выбирете необходимые параметры, см. images/image_2, images/image_3. Файл(.clang-format) со стилем кода должен находится в папке проекта.
3. Выбор автоформатирования кода: при сохранени файла(см. images/image_4), по горячей клавише(см. images/image_5).
