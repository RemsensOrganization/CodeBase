# C++
Правила и утилиты для автоформатирования кода С++ в проектах CodeBase

## Настройка_и_установка
1. Включить необходимые модули(Beautifier и ClangFormat), см. рис. 1.
![Скриншот окна "Параметры" для включения модулей.](images/image_1.png)
*Рисунок 1 - включение модулей.*
2. Появится модуль "Стилизатор", выберите необходимые параметры(Включить автоформатирование ClangFormat, выбрать путь к clang-format.exe и file со стилем кода), см. рис. 2, рис. 3. Файл(.clang-format) со стилем кода должен находится в папке проекта.
![Скриншот окна модуля "Стилизатор"](images/image_2)
*Рисунок 2 - включение автоформатирования ClangFormat.*
![Скриншот окна модуля "Стилизатор"](images/image_2)
*Рисунок 3 - установка автоформатирования ClangFormat.*
3. Выбор автоформатирования кода: при сохранении файла(включить переопределение файла ClangFormat, см. рис. 4), по горячей клавише(бинд клавиши для команды FormatFile, см. рис.4).
![Скриншот окна выбора стиля кода](images/image_4)
*Рисунок 4 - выбор стиля кода ClangFormat.*
![Скриншот окна бинда горячей клавиши](images/image_5)
*Рисунок 5 - использование ClangFormat по горячей клавише.*
