# Test Generator Module

Модуль предназначен для создания шаблонных тестовых проектов Qt (каталог + .pro, .h и .cpp файлы).

## Возможности

- Выбор целевой директории через дерево `QFileSystemModel`.
- Создание новой подпапки для теста с проверкой имени (запрещены `\ / : * ? " < > |` и управляющие символы).
- Генерация файлов проекта по заранее заданным шаблонам (`kProTestText`, `kHeaderTestText`, `kCppTestText`).

## Интерфейс

- Поле ввода имени теста с валидацией.
- Дерево каталогов для выбора целевой папки.
- Кнопка «Создать тест» для генерации структуры.
- Кнопка «Создать папку» для добавления подкаталога в выбранной директории.

## Использование

1. Выберите корневую папку через контекстное меню дерева.
2. Введите имя теста в поле и нажмите «Создать тест».
3. При необходимости создайте вложенную папку кнопкой «Создать папку».

## Требования

- Qt 5/6 с модулями Widgets и Gui.
- Права на запись в выбранную директорию.

## Структура создаваемого теста

- `<ИмяТеста>/` — каталог теста.
- `<ИмяТеста>.pro` — файл проекта Qt.
- `<ИмяТеста>.h` — заголовок класса теста.
- `<ИмяТеста>.cpp` — реализация класса теста.
